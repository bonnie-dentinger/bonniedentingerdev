I developed the PrismX website as part of my Web Programming class project, showcasing my proficiency in HTML, CSS, JavaScript, FuelPHP, and MySQL. PrismX is a fictional company aimed at assisting users in identifying color-blindness through interactive exercises. The website incorporates an API to generate color names dynamically. Additionally, users can access a feature allowing them to view a "printable" version of their creations. This project exemplifies my skills in full-stack web development and highlights my ability to integrate external APIs for enhanced functionality.

I implemented a dynamic customization feature on the PrismX website, enabling users to define the layout with a chosen number of columns and rows (ranging from 1 to 26) and select from a palette of up to 6 colors. The system incorporates meticulous validation checks to ensure the accuracy of user inputs. Any invalid entries prompt user-friendly alerts, reinforcing the commitment to a seamless and error-free experience. This project exemplifies my dedication to crafting interactive and user-centric web applications.

Furthermore, users have the capability to individually color the squares within the generated grid above. The grid size aligns with the user's selected number of columns and rows. This interactive feature allows users to click on squares and choose a specific color for them. If a color is modified, the colored squares seamlessly transition to the updated hue. Conversely, deleting a color results in the corresponding squares becoming uncolored. This functionality enhances user engagement by providing a dynamic and customizable experience.
