This website was a project for my Web Programming class. It is a website for a fictional company called PrismX, which helps provide exercises that help with identifying color-blindness. The website is built using HTML, CSS, JavaScript, FuelPHP, and MySQL and uses an API to generate color names. There is also functionality that allows the user to view a "printable" version of what they have created.

The user starts by choosing any number of columns and rows (between 1 and 26), as well as colors (between 1 and 6). There is validation built in to verify that the user has entered a valid number and alerts the user if there is invalid entry.

The user can also select a specific color, and "color" the squares created in the grid below, which was generated using the number of columns and rows selected by the user, by clicking on the squares. If the color is edited, then the colored squares will change to the new color. If the color is deleted, then the colored squares will empty.
